Personal Site

Coded from scratch with HTML, CSS, and JavaScript, this site acts as a home page for my portfolio and other website profiles. I used the opportunity to experiment with animations and effects like CSS perspective for a parallax effect. The design, prototyped in Adobe XD, combines my love for hiking and the night sky.

Study Abroad Matching (SAM)

SAM is a group collaboration to create an automated interface to match students with their preferred foreign-exchange programs. Given input parameters such as GPA, language certifications, and order of preference, SAM generates lists of students for each program based on a greedy algorithm. SAM primarily uses React, Google Cloud Platform, Django, and Python.

Logic 2020

Logic 2020 is a web implementation of Logic 2010, a digital workbook used to practice logic questions. Logic 2020 uses MongoDB and Express to provide both a student and teacher interface that gives immediate feedback for questions and shows student completion and accuracy.

Team Garbage

Team Garbage is an app prototype with the goal of teaching people proper recycling rules in Toronto. Following extensive user testing such as interviews, surveys, and observation, my group designed the app experience to make recycling quick and rewarding to understand.

GameCentre

GameCentre is an Android game app developed in Java with Android Studio. The app features four different games with independent leaderboards and save states. Development focused on using design patterns such as states and abstract factories.

Musichar/Playlistats

Musichar is an in-progress web app that calculates statistics on Spotify playlists and generates a character based on those statistics. Using Spotify's API, users can see statistics such as energy and "happiness". which will also be traits of the character in the generated image. Playlistats is the in-progress JavaScript library to calculate the necessary statistics.
